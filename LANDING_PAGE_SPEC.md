## Sections (in order)

### 1. Hero Section

Navigation bar with: How It Works | Features | Pricing | Pilot

### 2. Pain Section
Property managers are stuck chasing contractors for updates while boards ask for proof, budget status, photos, and payment explanations.

Vendor invoices arrive in one place. Photos arrive by text. Contracts live in another folder. Board updates are rebuilt manually before every meeting.

VendorProof HOA organizes contractor progress, invoice backup, project milestones, and budget status into one simple report.

Use icons or visual markers for each pain point.

### 3. Solution Section (How It Works - 5 Steps)

#### Step 1 — Create the Project
Enter the community, vendor, approved budget, contract amount, and project milestones.

#### Step 2 — Send Contractor Upload Link
The contractor receives a simple text or email link. No app download required.

#### Step 3 — Collect Progress Proof
The contractor uploads photos, notes, invoices, and completion status from the field.

#### Step 4 — Compare Billing to Progress
The manager sees project completion, invoice amount, budget remaining, and any variance concerns.

#### Step 5 — Export Board Report
Generate a clean PDF report for board meetings, payment approvals, and project records.

Each step should have a simple icon/illustration.

### 4. The "Gotcha" Visual Flow
Show this as a clear 4-step visual:
1. Contractor receives upload link
2. Contractor uploads photos and invoice
3. Manager sees progress vs billing
4. Board-ready PDF is created

### 5. Features Section

Display as a grid of feature cards with icons.
